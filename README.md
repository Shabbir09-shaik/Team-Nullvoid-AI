LearnSphere – AI Powered Chat Web Application

LearnSphere is a simple AI-powered web chat application built using Flask (Python), HTML, CSS, and JavaScript.  
It provides a clean chat interface with typing animation and chat history functionality.

Project Overview:

LearnSphere is designed to demonstrate:

- Backend development using Flask
- Frontend development using HTML, CSS, JS
- Client-server communication using Fetch API
- Chat UI with typing animation
- Session-based chat history

This project is developed as part of academic learning in B.Tech CSE.

Technologies Used:

1. Python 3.9
2. Flask
3. HTML5
4. CSS3
5. JavaScript
6. VS Code

Installation & Setup:

Step-1: Clone the Repository
git clone <your-github-repo-link>
cd nullvoid


Step 2: Create Virtual Environment

python -m venv venv


Step 3: Install Dependencies

venv\Scripts\python.exe -m pip install -r requirements.txt


Step 4: Run the Application

venv\Scripts\python.exe app.py


Step 5: Open in Browser
The URL shows like this http://127.0.0.1:5000/ in the VS Code Terminal
